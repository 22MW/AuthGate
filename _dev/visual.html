Resumen humano

AuthGate mantiene release 1.1.1 publicada y prepara el siguiente bloque como piloto de backend visual 22mw-back. La Fase 1 queda definida como shell admin, menú horizontal tipo landing y dark/light por localStorage, sin tocar frontend ni lógica funcional.

Fase 1 - Scope aprobado

  • Shell visual admin 22MW.
  • Header con branding AuthGate/22MW.
  • Menú principal horizontal, inspirado en landing/web.
  • Dark/light con localStorage.
  • Tabs siguen por URL ?tab=; AJAX queda para Fase 4.
  • No aplicar todavía a pantalla site-level multisite.

No tocar en Fase 1

  • Frontend público.
  • Lógica de login/registro.
  • Guardados existentes.
  • AJAX nuevo de tabs.
  • Versionado/release.
